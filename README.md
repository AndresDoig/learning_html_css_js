## 📖 Descripción

Este proyecto es un **sandbox educativo** diseñado especialmente para principiantes en desarrollo web. Incluye ejemplos prácticos y bien documentados de HTML, CSS y JavaScript que puedes modificar y experimentar para aprender haciendo.

### 🎯 Objetivo

Proporcionar un entorno seguro y completo donde los estudiantes puedan:
- Experimentar con código sin miedo a "romper" nada
- Ver resultados inmediatos de sus cambios
- Aprender conceptos fundamentales con ejemplos prácticos
- Progresar desde lo básico hasta conceptos intermedios

## 💻 Instalación

### Opción 1: Descarga Directa
1. Descarga los archivos del proyecto
2. Guarda cada archivo en la misma carpeta:
   - `index.html`
   - `styles.css`
   - `script.js`
3. Abre `index.html` en tu navegador

### Opción 2: Usando Git
```bash
# Clona el repositorio
git clone [URL-del-repositorio]

# Navega al directorio
cd html-css-js-sandbox

# Abre index.html en tu navegador
```

## 📁 Estructura del Proyecto

## 🚀 Uso

### Inicio Rápido
1. **Abre VS Code**
2. **Abre la carpeta** del proyecto
3. **Instala Live Server** (extensión)
4. **Click derecho** en `index.html` → "Open with Live Server"
5. **¡Empieza a experimentar!**

### Flujo de Trabajo Recomendado
```
1. 👀 Observa el código existente
2. 🔍 Lee los comentarios explicativos
3. ✏️ Haz pequeños cambios
4. 🔄 Observa los resultados en el navegador
5. 🧪 Experimenta con nuevas ideas
6. 📚 Consulta documentación cuando necesites
```
